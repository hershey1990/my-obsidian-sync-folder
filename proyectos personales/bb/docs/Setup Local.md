# BB — Setup Local

> Documento en borrador. Se completará al iniciar el desarrollo.

## Requisitos

- Node.js 20+
- pnpm
- PostgreSQL 15+ (o Supabase local)
- Git

## Levantar el proyecto

```bash
# Backend
cd apps/api
pnpm install
pnpm dev

# Frontend
cd apps/web
pnpm install
pnpm dev
```
