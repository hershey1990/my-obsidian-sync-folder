# Análisis Competitivo

## Mapa competitivo

| Competidor | Tipo | Fortalezas | Debilidades | Amenaza |
|------------|------|------------|-------------|---------|
| **Facebook Marketplace** | Clasificados general | Alcance masivo (~60% share), gratis, ya tiene usuarios | No especializado, UX pobre para autos, sin confianza, búsqueda genérica, estafas | 🟡 Media |
| **Encuentra24** | Clasificados general | Marca conocida en Centroamérica, tráfico estable | UX anticuada, mobile horrible, spam, publicidad excesiva, sin diferenciación | 🟢 Baja |
| **Kash** | Pagos P2P | Ya tiene usuarios, infraestructura de pagos | No es competitivo directo (es pagos, no autos) | 🟢 Ninguna |
| **Grupos de WhatsApp/Facebook** | Comunidad | Nicho, confianza de grupo | No escala, no hay búsqueda, manual | 🟢 Baja |
| **Lotes de autos / dealers** | Físico | Ver el auto en persona, confianza local | Solo local, poca visibilidad digital, inventario limitado | 🟡 Media |
| **Recomendación boca a boca** | Informal | Alta confianza | No escala, lento | 🟢 Baja |

## Diferenciación de Estela

### vs Facebook Marketplace
| Dimensión | Facebook | Estela |
|-----------|----------|--------|
| Búsqueda de autos | Genérica (todo el Marketplace) | Especializada, filtros reales de auto |
| Calidad de listings | Fotos malas, descripciones pobres | Checklist estandarizado + score |
| Confianza | Baja (perfiles falsos, estafas) | Identidad verificada, reputación |
| Comparación | No existe | Modelos + ofertas lado a lado |
| Dealers | No diferenciados | Perfiles especializados, badges |
| Inspección | No | Score digital + red de inspectores |

### vs Encuentra24
| Dimensión | Encuentra24 | Estela |
|-----------|-------------|--------|
| UX / UI | 2010, lento, invasivo | Moderno, rápido, mobile-first |
| Filtros de auto | Limitados | Especializados |
| Spam / duplicados | Alto | Controlado por diseño |
| Score de condición | No | Sí |
| Comparación | No | Sí |

## Barreras de entrada para competidores

1. **Catálogo de modelos**: Construir DB de specs de autos es tedioso (pero se hace una vez)
2. **Red de inspectores**: Requiere tiempo y confianza — ventaja del fundador que ya inspecciona
3. **Efecto de red**: Más autos → más compradores → más vendedores (difícil de arrancar pero sólido una vez en movimiento)
4. **Trust**: Reputación de vendedores es difícil de transferir a otra plataforma

## Nuestra ventaja sostenible

> **El fundador es un experto en autos que ya hace inspecciones.** Nadie más en Nicaragua combina:
> - Pasión + conocimiento real de autos
> - Habilidad técnica para construir la plataforma
> - Visión de crear un estándar de calidad

Esto no es un clasificados más. Es una plataforma construida por alguien que entiende y ama los autos.
