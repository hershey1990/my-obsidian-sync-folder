# ADR-003: Mercado Objetivo

## Contexto
Necesito definir el mercado objetivo para enfocar la búsqueda laboral: geografías, tamaño de empresa, roles target, salary bands, y canales de aplicación. Sin esta definición, la búsqueda es dispersa y poco efectiva.

## Decisión

### 1. Geografías Priorizadas

| Prioridad | Mercado | Razón |
|---|---|---|
| 1 | **LATAM remoto** | Idioma nativo, husos compatibles, alta demanda de seniors con AI skills |
| 2 | **España remoto** | Misma lengua, huso compatible, mercado tech activo |
| 3 | **USA remoto** | Cliente actual, inglés profesional, pero menor prioridad por competencia y huso |

### 2. Tamaño de Empresa

**Startups < 50 personas** exclusivamente.

Preferencia por early-stage a growth-stage donde el impacto individual es mayor y la jerarquía mínima.

### 3. Roles Target

Se aplica a cualquiera de estos títulos:
- Senior AI Software Engineer
- Senior Software Engineer (AI-Enabled)
- Staff Software Engineer + AI
- AI Solutions Architect
- Cualquier variante que combine senior SWE + AI/ML integration

El filtro real no es el título exacto sino: **rol senior de ingeniería donde se valore la integración de IA en el ciclo de desarrollo**.

### 4. Salary Target

**Mínimo: $4,500 USD/mes** (~$54K/año).

Justificación:
- LATAM senior AI engineer range: $4K-$8K/mes
- España senior range: €50K-€80K/año
- USA remoto range: $8K-$15K/mes (oportunidad si surge)
- $4,500 es el piso mínimo, no el target aspiracional

### 5. Industrias

Abierto. Sin restricción vertical. Startups de cualquier industria son bienvenidas.

### 6. Canales de Búsqueda

| Canal | Propósito |
|---|---|
| LinkedIn (jobs + networking) | Principal canal de aplicaciones |
| We Work Remotely / Remote OK | Startups remote-first |
| Y Combinator Work at a Startup | Startups early-stage |
| Otta (español/europeo) | Mercado España/Europa |
| AngelList / Wellfound | Startups US remote-friendly |
| Referrals / red propia | Alta tasa de conversión |

## Alternativas Consideradas

| Alternativa | Pros | Contras |
|---|---|---|
| Solo USA remoto | Mayor salario potencial | Mayor competencia, huso complicado |
| Solo LATAM | Idioma, cultura | Menor densidad de startups AI-enabled |
| Empresas grandes (>1000) | Estabilidad, proceso estructurado | Menor impacto individual, burocracia |

## Consecuencias
- Positivo: Enfoque claro, evita dispersión
- Positivo: Prioriza geos donde soy más competitivo (idioma, huso)
- Riesgo: Startups <50 personas pueden tener menor estabilidad (mitigación: evaluar runway en cada aplicación)
- Riesgo: LATAM tiene menos postings que USA (mitigación: compensar con networking activo)
- Acción: Crear alertas en LinkedIn y plataformas para los roles target en LATAM + España
