# TSK-003: Design system (shadcn/ui + layout base)

Configurar el sistema de diseño en el monorepo.

## Entregables
- shadcn/ui init con tema personalizado (colores Estela)
- Componentes base: Button, Input, Card, Dialog, Select, Badge
- Layout base: Header con logo + nav, Footer
- Tipografía y espaciado consistentes
- Modo responsivo mobile-first

## Criterios de aceptación
- Los componentes renderizan en web y admin
- El layout se ve bien en mobile y desktop
- Tema oscuro/claro funcional
