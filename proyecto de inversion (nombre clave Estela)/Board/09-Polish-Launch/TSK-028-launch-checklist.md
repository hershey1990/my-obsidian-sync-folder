# TSK-028: Launch checklist

Preparativos finales para el lanzamiento público.

## Entregables
- Dominio configurado (estela.com.ni o similar)
- SSL/HTTPS funcional
- DNS apuntando a Vercel + DigitalOcean
- Monitoreo de uptime (UptimeRobot o similar)
- Términos y condiciones + Política de privacidad
- Página de "Contacto" y "Acerca de"
- Landing page con SEO final
- Cuentas de redes sociales creadas (IG, FB)
- Plan de lanzamiento: comunicación, mailing lista early access
- Backup automático de base de datos configurado

## Criterios de aceptación
- estela.com.ni carga en HTTPS
- Términos y condiciones accesibles desde footer
- Backup automático diario configurado
- Plan de lanzamiento documentado y listo para ejecutar
