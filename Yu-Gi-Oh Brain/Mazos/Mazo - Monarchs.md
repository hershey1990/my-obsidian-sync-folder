## Combos principales
1. **Standard Domain lock**: NS Edea → SS Eidos → +1 Normal → Tribute Edea+Eidos for Ehther → Activate Domain → search Erebus → pass. Domain + Ehther (pop 1) + Erebus en mano
2. **Pantheism setup**: Activate Pantheism → draw 2 → dump Ehther/Erebus → Tenacity search for Domain/Stormforth
3. **Stormforth + Ehther**: Stormforth tributa monstruo rival → Ehther → pop dominio → set Return of the Monarchs
4. **Vanity's lock**: Vanity's Fiend + Domain = rival no puede SS ni usar extra deck

## Tech choices
- **Domain of the True Monarchs**: Core del mazo, reduce tributos y lockea extra deck
- **Vanity's Fiend/Majesty's Fiend**: Lock adicional que complementa Domain
- **The Monarchs Erupt**: Skill Drain versión monarca (solo afecta si controlas tribute summoned)
- **One for One**: Busca Edea, arranca el combo

## Matchups
- **Vs Yubel**: Favorable — Domain lock + Ehther pop rompe su board
- **Vs Snake-Eye**: Favorable si Domain se activa temprano (no pueden extra deck)
- **Vs Tenpai**: Neutral — Domain no frena Tenpai (no usan extra deck), pero Stormforth + Ehther pueden romper su field
- **Vs Labrynth**: Desfavorable — Domain no afecta, y Archfiend Eccentrick puede pop Domain
