## Combos principales
1. **Standard DM setup**: NS Magician's Rod → search Soul Servant → Soul Servant search Dark Magician → set Navigation/Eternal Soul → pass with DM + Circle + Navigation
2. **Magician's Souls**: Discard DM → draw 2, dump DM → Soul Servant recover DM → Eternal Soul revive
3. **Dragoon**: Activate The Eye of Timaeus targeting Dark Magician → summon Red-Eyes Dark Dragoon (negate + pop + burn)
4. **Super Poly**: Use opponent's monster + DM for The Dark Magicians

## Tech choices
- **Skill Drain**: DM no usa efectos en campo, Skill Drain no afecta jugabilidad
- **Super Polymerization**: Rompe boards ajenos, targets con Dark Magician
- **The Eye of Timaeus**: Acceso a Dragoon sin engine extra

## Matchups
- **Vs Snake-Eye/Fire King**: Favorable si abre Circle + Eternal Soul (banish Diabellstar)
- **Vs Tenpai**: Desfavorable, no puede seguir el ritmo
- **Vs Labrynth**: Favorable, Secret Village y Skill Drain los frenan
