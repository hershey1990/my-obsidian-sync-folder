# Dark Magician + Monarchs

## Premisa
Combinar el engine de Dark Magician (tribute summons de DM con Eternal Soul) con Domain of the True Monarchs y Edea/Eidos. La idea: DM da control y recovery, Monarchs da Domain lock + monarcas de alto impacto. Edea puede tribute summon Dark Magician desde el deck sin consumir Normal Summon.

## Core combo
1. NS Edea → SS Eidos → +1 Normal → Tribute Eidea+Eidos for Dark Magician
2. Activate Domain → lock extra deck rival + reducir tributo de Ehther/Erebus
3. Dark Magical Circle set → banish algo del rival
4. Eternal Soul revive DM cada turno
5. Ehther tribute summons sobre DM para pop, Erebus shuffle

## Viability

### Por qué sí
- Domain lock rompe la mayoría del meta actual
- DM y Monarchs comparten atributo oscuro/lanzador
- Soul Servant puede buscar cualquier carta DM, incluyendo Eternal Soul y Dark Magical Circle
- Tribute summon de Ehther sobre DM es un +1 neto

### Por qué no
- Dos engines grandes → más dead hands
- Sin Pantheism o Soul Servant la mano puede estar muerta
- Monarchs requiere Edea/Eidos, DM requiere Magician's Rod — competencia por NS
- Sin espacio para handtraps suficiente

## Notas adicionales
Probar en evento limitado primero. No recomendar para ranked competitivo. Alternativa: reducir Monarchs a engine mínimo (Domain + 2 Ehther + 1 Erebus) solo para el lock.
