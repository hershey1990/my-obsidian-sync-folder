# DOX — Ideas

## Purpose
Ideas de decks generadas por IA o por el usuario. Espacio para theorycrafting antes de armar el mazo.

## Ownership
- Todos los archivos `Idea - *.md`

## Frontmatter requerido
```
tipo: idea
nombre: string
arquetipos_base: [] (arquetipos involucrados)
origen: ia | propia | teoria
estado: pendiente | probar | descartada | adoptada
fecha: YYYY-MM-DD
tags:
  - ygo/idea
```

## Contenido esperado
- **Premisa**: qué busca lograr la idea
- **Core combo**: la interacción clave
- **Deck list sugerida**: al menos main deck conceptual
- **Viability**: qué tan viable se ve (por qué sí / por qué no)

## Workflow de ideas IA
1. Leer arquetipos disponibles en la base
2. Identificar sinergias potenciales entre engines
3. Generar idea con estructura completa
4. Crear nota y dejar estado: pendiente
