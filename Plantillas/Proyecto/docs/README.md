# docs/

Documentación pulida para exportar a Outline.

Agregar archivos `.md` con frontmatter:

```yaml
---
title: "Título del documento"
description: "Descripción para Outline"
actualizado: YYYY-MM-DD
outline_status: pendiente
outline_url: null
---
```

Cuando se publique en Outline, cambiar `outline_status: publicado` y llenar `outline_url`.
